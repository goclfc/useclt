useclt is a bot that simplifies your deployment workflow. Push, build, and ship your apps without the hassle.

⚡

Zero Config

Automatically detects your framework and configures the build. No YAML files, no setup wizards.

🚀

Instant Deploys

Push to deploy in seconds. Every branch gets a unique preview URL for easy testing and reviews.

🔒

Built-in SSL

Automatic HTTPS for every deployment. Custom domains with free SSL certificates, no extra steps.

🤖

Bot-Powered

Integrates with GitHub, GitLab, and Slack. Get deploy notifications and trigger rollbacks via chat.

🌏

Edge Network

Your app is served from the edge, close to your users. Fast load times everywhere in the world.

📈

Logs & Metrics

Real-time logs, deploy history, and performance metrics. Know exactly what's happening at all times.

How it works

01

Install the CLI

Run npm i -g useclt and authenticate with your account.

02

Link your project

Run clt init in your project root. We detect the framework automatically.

03

Deploy

Run clt deploy and you're live. Set up Git hooks for automatic deploys on push.
